# Restaurant Website (POS + CRUD)

## How to run

- Open `restaurant-website/index.html` in Chrome/Edge.
- No server required.

## What’s included

- **Menu (with images)**: click any item to add to cart.
- **Billing**: subtotal + optional tax %, total auto-calculates.
- **Cart controls**: increase/decrease qty, remove item, clear cart.
- **Pay now (QR)**: generates a UPI QR code.
- **Print bill**: print-friendly layout.
- **Monthly sales report**: saved orders + totals by month.
- **Manage menu (CRUD)**: add/edit/delete items, set active/inactive, set image URL.

## Notes

- Data is stored in your browser using `localStorage`.
- Update the UPI ID in `app.js` under `DEFAULT_SETTINGS` (or when prompted on first QR open).
